Guía práctica de comandos de GIT

Git, un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

En este documento dejamos una lista de los comandos más utilizados por desarrolladorxs y usuarios.

Recordá que siempre podés escribir git help

CREAR REPOSITORIO

git init

git clone URL

VER ESTADO DE REPOSITORIO

git status

HACER COMMIT

git add .

git commit -m "comentario descriptivo"

GUARDAR EN REPOSITORIO REMOTO

git init

git clone URL
